# flutterapp

采用 flutter 做移动端开发 <br/>
采用 vue + vuex + vue-router + axios + ts 做后台发布系统<br/>
采用 eggjs + mysql 做后台模拟操作<br/>

体验从移动端到后台管理系统的整个开发流程，同时可以做为自己的工作笔记记录下来，随时翻阅
